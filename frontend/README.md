# frontend

React 18 / Vite / TypeScript / Tailwind CSS — advertiser dashboard.

## Run locally

```bash
npm install
npm run dev   # http://localhost:3000
```

## Key libraries
- **React Query** — server state, caching, background refetch
- **Zustand** — client state (auth token, UI state)
- **React Hook Form + Zod** — form validation
- **Recharts** — campaign metrics charts
- **React Router v6** — client-side routing
